Setup instructions

The easiest way to try out F* quickly is directly in your browser using the online editor that's part of the F* tutorial.
(We also have an even cooler online editor (experimental).)

For people who want to get F* (v0.9.6.0) installed on their machine there are 2 easy options:

  1. Using the binary release (v0.9.6.0) (documentation on this)
  2. or Using the OPAM package (especially for people already using OCaml)

In addition, if you're familiar with emacs, we recommend getting fstar-mode.

For people who are more motivated, there are even more ways including building things from sources (see INSTALL.md).
